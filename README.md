# 🧺 Aplikasi Laundry - Mobile App (Android)

Selamat datang di dokumentasi tampilan *Aplikasi Laundry*!  
Aplikasi ini dibuat untuk memudahkan pengelolaan transaksi laundry, pelanggan, pegawai, layanan, dan laporan secara mobile dengan antarmuka modern dan mendukung **mode terang & gelap**.

## 🚀 Teknologi
- **Bahasa**: Kotlin
- **Database**: Firebase Realtime Database
- **UI**: Material Design
- **Fitur**: CRUD, autentikasi, laporan dinamis, dan banyak lagi.
